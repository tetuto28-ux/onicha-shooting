# Rojo Setup

This repository is configured for Rojo live sync.

## Start Rojo

Run:

```bat
run-rojo.cmd
```

Rojo serves this project at:

```text
localhost:34872
```

## Connect From Roblox Studio

1. Open Roblox Studio.
2. Open the Rojo plugin.
3. Connect to `localhost:34872`.

Synced services:

- `ReplicatedStorage`
- `ServerScriptService`
- `StarterGui`
